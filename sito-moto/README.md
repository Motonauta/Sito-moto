# Sito Moto — Nome Cognome

Sito statico (HTML/CSS/JS) pronto per essere pubblicato su Vercel tramite GitHub.

## Struttura
- index.html — Home
- sponsor.html — Sponsor
- galleria.html — Galleria
- contatti.html — Contatti
- style.css — stile condiviso
- script.js — interazioni condivise

## Pubblicazione
Segui la guida ricevuta in chat: crea un repository GitHub, carica questi file,
poi collega il repository a Vercel per il deploy automatico.

## Immagini da Cloudinary
Nei file galleria.html e index.html, sostituisci i blocchi segnaposto
(<div class="gallery-item" ...>) con tag <img src="URL-CLOUDINARY">
una volta caricate le foto reali su Cloudinary.
